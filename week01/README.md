## рЯТ° Key Takeaways

- Python uses **indentation** to define blocks вАФ 4 spaces is standard
- `for...else` and `while...else` are unique to Python вАФ `else` runs only if no `break` occurred
- Functions without a `return` statement return `None` by default
- **Lists** are mutable; **Tuples** are immutable вАФ choose based on whether data should change
- The `if __name__ == "__main__"` guard prevents module code from running on import
- List comprehensions are more Pythonic and readable than equivalent `for` loops
